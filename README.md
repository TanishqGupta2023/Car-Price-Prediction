# Car Price Prediction

This project predicts the selling price of used cars using machine learning.
It is implemented in a Google Colab notebook.

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib

## Models Used
- Linear Regression
- Random Forest Regressor

## Dataset
- Source: Kaggle
- Size: 121 rows

## How to Run
Open the `.ipynb` file in Google Colab and run all cells.
